Latest changes

Here's what was new in September/October of 2021:

  • Added the page "What's missing from your Swagger docs . . ."

  • Edited and updated the existing pages.

  • Added Google Analytics.

Here is what was new as of July/August 2021:

  • Added the section called "The API reference docs: What to include."

  • Added an example file to demonstrate the content of reference documentation for a single API endpoint.

Here is what was new on the site, as of June 2021:

  • Added a new topic about the minimum viable documentation required for an API.

  • Added sections called "About this site" and "About the author."

  • Created subsections for developer and technical writer use cases.

  • Switched to a build method based on an Oxygen Lightweight DITA transform, hence the new design and new site architecture.

The toolchain

Here's are the tools used to edit source files and build the site.

Item Description
Deployment Netlify
Build/transform Oxygen XML Editor and out-of-the-box responsive web help transform with slight customizations.
Source code management GitHub
Editing tools Atom and Oxygen XML Editor (I own an individual license)
Markup language Lightweight DITA (mostly Markdown with HTML tables)

Find out more about Lightweight DITA from Oasis. Or Read Carlos Evia's book, "Creating intelligent content with Lightweight DITA".
